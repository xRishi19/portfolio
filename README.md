# Rishiraj Singh - Personal Portfolio

This repository contains my personal portfolio website, built with ReactJS and Vite.

## Tech Stack
- **Framework**: ReactJS
- **Build Tool**: Vite
- **Styling**: Vanilla CSS with a customized Dark Theme (Glassmorphism & Micro-animations)

## Key Sections
- **Hero**: Introduction and contact/resume links.
- **Experience**: A clean timeline layout detailing work at RTX, Virginia Tech, and BOLT.
- **Projects**: Highlights of technical projects like the Auto Delivery UAV System and Image-Based Group Sentiment Classification.
- **Education & Skills**: Overview of my B.S. in Computer Science at Virginia Tech and a categorized list of my technical languages and frameworks.

## Local Development
To run this project locally:

1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to `http://localhost:5173`.
